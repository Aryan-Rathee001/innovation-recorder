# Innovation Hub - Presentation Ready

## Overview
Innovation Hub is a web platform where users can submit, browse, and rate innovative ideas across different categories.

## Files Structure (Simplified for Presentation)

### Core Files
- **index.html** - Homepage with hero section and navigation
- **submit_idea.html** - Form to submit new ideas
- **view_ideas.php** - Browse and rate existing ideas
- **dashboard.php** - Analytics and statistics
- **rate_idea.php** - Handles rating submissions
- **submit_idea.php** - Handles idea submissions

### Configuration
- **db.php** - Database connection
- **database_schema.sql** - MySQL database structure

### Assets
- **style.css** - Simplified styling
- **script.js** - Search functionality

## Features
1. **Submit Ideas** - Users can share their innovative concepts
2. **Browse Ideas** - View ideas by category with search functionality
3. **Rate Ideas** - Community feedback system (1-5 stars)
4. **Dashboard** - Analytics showing ideas distribution and top-rated concepts
5. **Resource Links** - Each idea category includes relevant GitHub/Reddit resources

## Categories
- Technology
- Healthcare
- Education
- Finance
- Others

## Database Tables
- **ideas** - Stores all submitted ideas
- **ratings** - Stores user ratings and comments
